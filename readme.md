## 📌 项目简介

**CopyView** 用来记录我在日常开发、浏览网站、刷产品时遇到的  
**设计优秀 / 交互舒服 / 视觉有冲击力的页面**。

主要目的：

- 📷 记录好看的页面设计  
- 🧩 拆解页面布局与交互思路  
- 🛠️ 尝试用代码复刻或改造  
- 💡 作为以后项目的灵感参考库

- 后台管理系统 UI
- 官网首页 / Banner
- 表单设计
- 卡片 / 列表布局
- 动效 / 交互细节
- 赛博朋克 / 科技风 / 极简风 等视觉风格
